# EKOU Landing

Landing page en Astro que comunica la propuesta de valor de EKOU: consultoría, tecnología y acompañamiento práctico para empresas, emprendedores e inversionistas.

## Requisitos
- Node.js 18 o superior
- npm

## Comandos

```bash
npm install
npm run dev     # entorno de desarrollo
npm run build   # genera la versión estática
npm run preview # sirve la build para revisión
```

## Estructura
- `src/pages/index.astro`: página principal con secciones de servicios, valores, ecosistema y casos de éxito.
- `src/components`: tarjetas reutilizables para servicios, valores, casos y filiales.
- `public/styles/global.css`: estilos globales y sistema visual.
- `public/favicon.svg`: ícono base.

## Contexto de contenido
La landing resume los servicios de consultoría, productos tecnológicos, valores de marca, ventajas competitivas, filiales del ecosistema y casos de éxito destacados.
